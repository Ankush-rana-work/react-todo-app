
# React Todo APP

ToDo List App is a kind of app that generally used to maintain our day-to-day tasks or list everything that we have to do, with the most important tasks at the top of the list, and the least important tasks at the bottom. It is helpful in planning our daily schedules.

## Demo

https://react-todo-app-86014.web.app/


## Installation

Run below command

```bash
  git clone https://github.com/Ankush-rana-work/react-todo-app.git
```
Now go to your project directory
```bash
  cd project-name
```
Run below command to install all dependency packages
```bash
  npm install
```
Run the project as below command
```bash
  npm start
```
